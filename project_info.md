# Архитектура и Дерево Проекта AI_HUB

## 1. Описание проекта
**Название:** AI Bottleneck — Многоканальный ИИ-Анализатор.
Это веб-приложение на базе Streamlit, предназначенное для получения максимально точных ответов на запросы пользователя, защищенных от ИИ-галлюцинаций. Приложение отправляет запрос пользователя сразу в 3 различные нейросети (работают параллельно), после чего передает их ответы 4-й нейросети ("Аналитику"), которая анализирует их на наличие ошибок и синтезирует финальный структурированный ответ.

## 3. Технологический стек и Архитектура
* **Фронтенд:** Streamlit (`app.py`). Используется кастомный CSS-код (эффекты "glassmorphism", неоновые градиенты, темная тема).
* **Бэкенд:** Python. Для параллельного вызова сразу трёх ИИ-моделей используется `concurrent.futures.ThreadPoolExecutor` (до 3 воркеров), что исключает долгое ожидание.
* **Провайдеры ИИ (API):**
  * `groq` (Llama 3.3 70B, Llama 3.1 8B)
  * `openrouter` (DeepSeek R1, Mistral Nemo)
  * `google-generativeai` (Gemini 2.0 Flash)
* **Управление состоянием (Session State):** Встроенный `st.session_state` используется для хранения истории текущей переписки, сохранённых чатов (сессий), выбранного языка и состояния интерфейса.

## 4. Промпты и Логика ИИ
Запрос пользователя отправляется в первые 3 модели в чистом виде (роль `user`). Главный промпт приложения встроен в логику "Аналитика", который делает финальный синтез:

**Системный Промпт Аналитика:**
> "Review the following 3 answers to: '{user_input}'. Point out any errors directly (e.g. 'Model 1 hallucinated...'). Synthesize the best parts into a final structured answer.
> 
> Win1: {a1}
> 
> Win2: {a2}
> 
> Win3: {a3}"

## 5. Доступные нейросети (Роли по умолчанию)
В системе зарегистрированы следующие модели:
* **Окно 1 (По умолчанию):** Llama 3.1 8B (Легкая / Groq)
* **Окно 2 (По умолчанию):** Mistral Nemo (Креатив, Free / OpenRouter)
* **Окно 3 (По умолчанию):** DeepSeek R1 (Логика, Free / OpenRouter)
* **Аналитик (По умолчанию):** Gemini 2.0 Flash (Быстрая / Google)
* *Доступна для выбора:* Llama 3.3 70B (Универсальная / Groq)

*Также реализована система **Fallback**-моделей (запасных) на случай, если ключи от Gemini или OpenRouter отсутствуют. Запросы в таком случае перенаправляются на бесплатные модели Groq.*

## 6. Основные функции и задачи в коде
* **Мультиязычность интерфейса:** Поддержка русского, английского и испанского языков (через словарь `T`).
* **История чатов:** Сохранение сессий и возможность переключения между старыми диалогами через боковую панель.
* **Интерфейс для медиа:** Есть кнопки прикрепления файлов и микрофона (`st.audio_input`), которые на данный момент выводят уведомление о "Демо-версии".
* **Отказоустойчивость:** Установлен таймаут на запросы к API (`PRIMARY_TIMEOUT = 45.0` секунд), чтобы приложение не зависало.
